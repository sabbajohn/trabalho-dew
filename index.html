JAVASCRIPT

ESCÂNDALO: Polícia do Código apreende quilos de JavaScript em site que prometia ser CSS puro!

Uma operação rotina na aba Network do navegador descobriu um arquivo oculto chamado script.js escondido atrás de um carrossel de cards.

O site alegava usar apenas inputs do tipo radio para fazer transições de página, mas os investigadores acharam funções addEventListener altamente perigosas rodando em segundo plano.

Semântica e Acessibilidade

<header>

É usado para representar o cabeçalho da página ou de uma seção.

<nav>

Representa a área de navegação, normalmente contendo menus e links.

<main>

Define o conteúdo principal da página.

<section>

Serve para agrupar conteúdos relacionados em seções.

<article>

Representa um conteúdo idependente, como notícias, posts ou artigos.

<aside>

É usado para conteúdos secundários, geralmente laterais.

<footer>

Representa o rodapé da página ou de uma seção.

<figure>

Agrupa imagens, gráficos ou outros elementos visuais.

<figcaption>

Define a legenda de uma figura ou imagem.

<time>

Representa datas e horários.

<mark>

Serve para destacar partes do texto.
